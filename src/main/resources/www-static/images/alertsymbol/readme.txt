be found at the site http://www.alertsymbol.com/ where more extensive
documentation is available.

The gif files were extracted in the zipfile at:

The file alertsymbol-technical-legend.pdf was generated from "Uppmärksamhetssignal legend.ppt" available in the zipfile.

The file alertsymbol-clinical-legend-sv.pdf is a renamed copy of the file "Symbol Uppmärksamhetsinformation_pdf.pdf"
available at https://sites.google.com/site/warninginformation/resurser-swe/SymbolUppm%C3%A4rksamhetsinformation_pdf.pdf?attredirects=0&d=1
