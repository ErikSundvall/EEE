openEHR archetypes define paths to data elements much like XPath. E.g. the path below:

/data/items[at0002]/value/defining_code/code_string
<----archetype----><-RM-><------data type--------->

describes the path to the code_string for the coded value of the tobacco usage status at node at0002 .

SNOMED CT code used: 8517006|Ex-smoker|.
