Developer view of major components

EEE is divided into some major components (see diagram) with an aim to have reusable structure in other implementations

For normal day-to-day clinical work. Login with user demo and password demo

  • /ehr/ and /ehr:{ehr-id}/ are used to read and query EHRs one at a time
    • /ehr/ is used e.g. to create new EHR ids. Autogenerated WADL interface descriptions for /ehr/ - [WADL/XML] [WADL as html]
    • for /ehr:{ehr-id}/ the {ehr-id} should be changed to a valid EHR ID, without the {} curly brackets. The /ehr:{ehr-id}/ format helps interface designers to easily make calls to openEHRs EHR URIs as described in Section 11.3 of the openEHR Architecture Overview by just prepending a slash / to an EHR URI it will thus become a fully functional http link to the desired EHR object. Developer tools and hints

Modern web browsers can be used to debug HTTP calls and JavaScript code (e.g. Firebug for Firefox or developer tools in Chrome). Most GUI-related code in LiU EEE do not require Java knowledge or compilation tools, instead they can be edited with text editors (preferrably ones supporting html and JavaScript) and a page reload is often enough to see the changed result. (Do not put your code under the /static/ directory during development if you want to avoid the browser shoeing old cached results.)

Instead of using an ordinary web browser you may want to do testing using a fully capable REST client like

  • a standalone, java based, GUI rest-client or
  • the RESTClient Firefox plugin, installation link: https://addons.mozilla.org/en-US/firefox/addon/9780

-This prototype installation by default has a http-query based tunnel service installed, that can be used as workaround for clients without full HTTP method support, by using query arguments like /ehr/trace/?method=OPTIONS&media=application/xml. (For POST requests, it lets you specify the actual method to use (DELETE, PUT, MOVE, etc.). For GET requests, let you specify OPTIONS as the actual method to use.) In a production environment you might want to adjust the tunneling settings to fit your setup.
